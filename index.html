XMLtoJSON API Documentation

  • Overview

These are the rules:

Request URL Parameters Response
GET /api ?feed=<feedUrl>
feed accepts the url of an XML feed (feedUrl) with ampersands delimeted		 Returns the original XML feed, translated into JSON
Note: Ampersands can be delimeted in the url by %26


Example:
GET /api?feed=
http://en.wikipedia.org/w/api.php
?format=xml
%26action=query
%26list=search
%26srsearch=wikipedia
%26srprop=timestamp
